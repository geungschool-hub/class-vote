# 학급 임원 선거 (class-vote)

학급회장·부회장 선거용 실시간 투표/개표 앱.

- `index.html` — 학생용 투표소 (핸드폰). QR(`?r=방코드`)로 접속 → 학번 확인 → 기표 → 투표함 투입. **후보가 1명이면 자동 찬반 투표.**
- `admin.html` — 교사용 개표 현황판 (전자칠판). **선거방 만들기(방마다 비밀번호)** → 선거 개설 → 실시간 투표함·QR → 개표(한 표씩 「正」자 집계) → 당선 발표.

**여러 반 동시 사용 가능**: 선생님마다 방(방 코드)을 만들면 `vote/rooms/{방코드}/`로 완전히 분리됨. 다른 선생님께는 admin.html 주소만 공유하면 됨.

백엔드: Firebase RTDB `vote/rooms/{rid}/` (gongju7-vocab, 무인증 공개경로 — 규칙 정본은 ai-tool `시험목표제출/database.rules.json`).

명부(voters)에는 학번·투표시각·표 키가 기록되고, 학번 삭제 시 해당 표도 함께 삭제됨(화면에는 표 내용·학번 연결이 표시되지 않음).
